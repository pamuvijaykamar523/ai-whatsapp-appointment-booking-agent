AI WhatsApp Appointment Booking Agent (n8n)

Overview

An AI-powered WhatsApp appointment booking system built using n8n.

Features

- AI-powered conversation
- Collects missing appointment details
- Memory-based follow-up
- Google Calendar integration
- WhatsApp confirmation messages
- Handles incomplete and complete bookings

Tech Stack

- n8n
- Google Gemini
- Twilio WhatsApp
- Google Calendar
- AI Agent
- Simple Memory

Workflow

WhatsApp → Webhook → AI Agent → Memory → IF Node → Google Calendar → WhatsApp Confirmation

- AI Automation
- Prompt Engineering
- Workflow Design
- API Integration
